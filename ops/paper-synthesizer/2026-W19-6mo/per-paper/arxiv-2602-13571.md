## Headline

LCR uses an LLM's own confidence — measured as Maximum Semantic Cluster Proportion across multinomial samples — as a training-free reranking signal: up to +20.6% NDCG@5 over fine-tuned rerankers using only a 7-9B LLM.

## Summary

The paper proposes a training-free, plug-and-play reranker that scores documents by an LLM's confidence in its answer with that document. Confidence is measured as Maximum Semantic Cluster Proportion (MSCP) across multinomial samples. The pipeline is two-stage: confidence assessment via sampling and clustering; binning + multi-level sorting based on query and document confidence thresholds, with high-confidence queries preserving original rankings. Evaluated on BEIR and TREC with BM25 and Contriever retrievers, LCR (with only 7-9B LLMs) consistently improves NDCG@5 by up to 20.6% across both pre-trained and fine-tuned reranker baselines.

For a practitioner: LCR is one of the earliest and cleanest expressions of the "use the LLM's confidence as the reranking signal, no training needed" approach that defines Phase 3 onward. The training-free property makes it easy to adopt — drop in any 7-9B LLM, no fine-tuning required. The "up to" qualifier in the headline number means the average gain is smaller, but the floor case (no degradation) is what makes it a low-risk pilot.

## Specifics

- Mechanism: LLM confidence via Maximum Semantic Cluster Proportion (MSCP) across multinomial samples.
