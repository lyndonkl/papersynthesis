# Pass A — Within-paper layered summary

## 30,000 ft (paper's claim, one sentence)
Decouples agentic-search training into two stages: stage-1 RL with decomposition-specific rewards for strategic planning, then stage-2 supervised fine-tuning on high-quality multi-hop trajectories for execution.

## 3,000 ft (argument + evidence, 2-3 sentences)
APEX-Searcher argues that end-to-end RL on agentic search suffers from ambiguous retrieval execution paths and sparse rewards, leading to inaccurate retrieval and degraded performance. The framework's solution is a two-stage decoupling: first RL with decomposition-specific rewards optimizes how the agent splits a multi-hop question into sub-tasks; built on the resulting decomposition, supervised fine-tuning on multi-hop trajectories teaches robust iterative sub-task execution. Across multiple benchmarks the framework reports significant gains in both multi-hop RAG and task planning.
