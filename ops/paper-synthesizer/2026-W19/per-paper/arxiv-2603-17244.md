## 30,000 ft (paper's claim, one sentence)
Grounds agent-memory architecture in formal AGM belief-revision postulates and demonstrates a graph-native memory (Redis + Neo4j + hybrid retrieval + client-side LLM rerank) that meets several K* / belief-base postulates while delivering 93.3% on LoCoMo-Plus.

## 3,000 ft (argument + evidence, 2-3 sentences)
Kumiho's central claim is a correspondence between the AGM belief-revision framework (revisions, retractions, expansions) and the operational semantics of a property-graph memory system, with proofs of basic AGM postulates K*2-K*6 and Hansson's belief-base postulates (Relevance, Core-Retainment). The architecture is dual-store (Redis working memory + Neo4j long-term graph) with hybrid full-text + vector retrieval and three architectural innovations: prospective indexing (LLM-generated future-scenario implications indexed at write time), event extraction (structured causal events in summaries), client-side LLM reranking. On LoCoMo: 0.565 F1 overall (n=1986) with 97.5% adversarial-refusal accuracy. On LoCoMo-Plus: 93.3% authors' result, mid-80s on independent reproduction — still well above the 45.7% best-published baseline.
