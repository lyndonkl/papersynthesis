# Pass A — Within-paper layered summary

## 30,000 ft (paper's claim, one sentence)
Recasts agentic dataset search as a software-architecture problem and offers a bounded, auditable reference architecture combining BM25 + dense retrieval + reciprocal-rank-fusion + LLM-agent reranking + offline metadata augmentation.

## 3,000 ft (argument + evidence, 2-3 sentences)
This paper addresses the vocabulary mismatch in dataset search, where natural-language queries fail against sparse provider-authored metadata, by introducing an offline step where an LLM generates pseudo-queries augmenting the retrieval indexes. Two architectural styles are compared: a single ReAct agent and a multi-agent horizontal architecture with feedback control; quality-attribute trade-offs are analyzed for modifiability, observability, performance, and governance. A seven-system-variant evaluation framework isolates the contribution of each architectural decision, with explicit governance tactics to bound and audit non-deterministic LLM components.
