### One-line summary
HCAG is a hierarchical retrieval-augmented generation framework for repository-level code generation in theory-driven codebases (e.g., algorithmic game theory), combining offline hierarchical abstraction, online hierarchical retrieval, and multi-agent discussion, with a claimed cost-optimality result and empirical gains on code quality and architectural coherence.
